## 📌 Overview

A **full-stack web application** built with Flask backend and interactive HTML/CSS/JavaScript frontend. Features user authentication, database management, and responsive design deployed on Vercel.

---

## ✨ Features

- 🔐 **User Authentication** – Login, signup, session management
- 📊 **Database Integration** – SQLite for persistent data storage
- 🎨 **Responsive UI** – Modern, clean interface
- ⚡ **Fast Performance** – Optimized backend with Flask
- 🚀 **Cloud Deployed** – Live on Vercel

---

## 🛠️ Tech Stack

**Backend:** Python, Flask  
**Frontend:** HTML, CSS, JavaScript  
**Database:** SQLite  
**Deployment:** Vercel

---

## 🚀 Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/aryanmandlik12/Neuralane.git
   cd Neuralane
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application:**
   ```bash
   python app.py
   ```

4. **Open in browser:**
   ```
   http://localhost:5000
   ```
